
# Playwright E2E tests

This repository contains end-to-end (E2E) tests built using Microsoft Playwright. It provides a scalable structure for UI test automation, including test reporting, environment management, and CI integration.

This project uses:
- Page Objects Model.
- Custom Fixture
- Cross-browser testing (Chromium, Firefox, WebKit) 
- Custom Typescript Linting rules

## Installation

```bash
  npm install
```
    
## Run tests

To run the tests 

```bash
  npm test
```


## Tech Stack

Playwright - TypeScript
